# 📊 Sales Forecasting System (Machine Learning)

## Overview
This project predicts future sales using machine learning models based on historical data.

## Features
- Sales prediction using ML algorithms
- Time series analysis
- Data visualization

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn / Statsmodels
- Matplotlib

## Results
- Improved prediction accuracy by 20–25%
- Helps in business decision-making

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python app.py
